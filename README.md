### 一.数字逻辑与电路设计实验
- 能够实现电机的正转、反转和暂停功能； 
- 用两个LED的三种状态代表电机的三种状态（例：全灭为暂停） 
- 一个工作周期包含正转25s、暂停5s、反转25s和暂停5s； 
- 可以自行设置工作的总时间（总时间需能够整除60s），时间使用数码管显示； 
- 工作时间结束后，数码管显示为0，蜂鸣器鸣叫5s。 
- 运用仿真软件对电路进行仿真

### 二.实验所用设备
